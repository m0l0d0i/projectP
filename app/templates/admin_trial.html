⚠️ Принудительный сброс trial для всех active-юзеров ?

Эта операция обнуляет users.trial_issued_at для всех active-юзеров (не заблокированы, не bot_blocked). Существующие подписки не трогаются — просто пользователь снова сможет нажать «Получить trial» в боте.

Реализовано через один SQL UPDATE, audit пишется единой строкой с total count. Roll-back возможен только через DB-restore.
