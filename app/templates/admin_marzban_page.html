Страница подписки Marzban ?

Управление публичной страницей подписки /sub/<token>. Здесь настраиваются тексты, platform-блоки, preview и live apply. Прямого raw-редактора .env здесь нет.

Live apply изменяет реальную страницу Marzban ?

Кнопка «Сохранить и применить в Marzban» пишет итоговый HTML в live deploy target, делает backup и при ошибке пытается выполнить rollback.

Если для apply настроена restart-команда Marzban/Xray, изменение может повлиять на live-отдачу страницы подписки. Перед apply проверьте preview, пути и итоговый diff.

Управляемые env-ключи меняются только по allowlist. Неизвестные ключи и raw-редактирование .env здесь недоступны.

ℹ️
Важно по пустым полям ?

Необязательные текстовые поля этой страницы можно очищать полностью: пустое значение сохраняется как NULL, а старый текст не подставляется молча.

Это относится к полям «Подсказка к подключению», «Текст блока поддержки» и «Подзаголовок блока платформ».

Для allowlisted env-ключей действует отдельный apply-flow: значения сначала проходят preview/валидацию и только потом пишутся в файл.

Сохранить и применить в Marzban ?

Эта операция рендерит live HTML, создаёт backup текущего deploy target, записывает новый файл атомарно и при неудаче пытается выполнить rollback.

Перед применением ?
  • • Убедитесь, что preview соответствует ожидаемому результату.
  • • Проверьте source template и deploy target выше.
  • • Если настроена restart-команда, apply повлияет на live-сервис Marzban/Xray.
